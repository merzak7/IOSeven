IOSeven

Est un système de supervision d'un ensemble de machines distantes (dans un réseau local ou internet) qui vous permet de garder un oeil proche sur toutes vos machines, de vous bien présenter les informations nécessaires pour savoir l'état de vos machines à n'importe quel instant.

Processes

You can use IOSeven to monitor daemon processes or similar programs running on system. IOSeven is particularly useful for monitoring daemon processes, such as those started at system boot time...

System

IOSeven can be used to monitor general system resources such as overall CPU usage, Memory and Load Average...

Files

You can also use IOSeven to monitor files, directories and filesystems. IOSeven can monitor these items for changes, such as timestamps changes, checksum changes or size changes.

Programs and scripts

IOSeven can be used to test programs or scripts at certain times, but in addition, you can test the exit value of a program and perform an action or send an alert if the exit value indicates an error. This means that you can use IOSeven to perform any type of check you can write a script for.
